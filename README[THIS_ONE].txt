What I am:
    The FA Referee Dashboard is a Python-based web application built with Flask that
    lets match officials record fixtures, log sanctions, view personal statistics,
    and let their admin access tools for appointemnts and development.
        Features:
            1. Secure referee registration and login system
            2. Add games with league and role information
            3. Record sanctions (yellow and red cards) against players
            4. Generate automatic referee statistics and performance summaries
            5. Search the 2025 IFAB Laws of the Game using keywords
            6. Analyse sanctions by team, league, or player
            7. Admin dashboard for referee officers to view statistics and referee details
            8. Simple, clean web interface styled with HTML and CSS style inspired by FA branding

My Requirments:
    Libraries: Flask, SQLite3, Jinja2
    Languages Used: Python v3.9 or later (backend), HTML (frontend), CSS (styling)
    SQLiteStudio, Pycharm, Web Browser

My Instillation Process:
    1. Download and Setup:
        Download the ZIP folder and decompress it onto your computer.
        Open the extracted project in PyCharm.
        Make sure all above requiremnts are installed.
        Open the schema.sql file and run it once to create the database structure.
    3. Run the app:
            Right-click app.py and select 'Run 'app".
            Click the IP address link in the console to open.
